# Instalação - Axios

Axios é um cliente HTTP baseado em Promises para fazer requisições a APIs no navegador e no Node.js.

## Comando Recomendado (npm)

Para o nosso boilerplate, usaremos `npm` como padrão.

```bash
npm install axios
```

### Alternativas

```bash
# yarn
yarn add axios

# pnpm
pnpm add axios
```
